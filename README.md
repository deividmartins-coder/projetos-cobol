# Sistema Bancário em COBOL

Este é um projeto de simulação de operações bancárias básicas desenvolvido em COBOL, ideal para prática de lógica de transações financeiras.

## Funcionalidades
- Cadastro de cliente e número da conta
- Consulta de saldo
- Depósito e saque
- Visualização dos dados da conta

## Tecnologias
- COBOL (GnuCOBOL / OnlineGDB)
- Programação procedural
- Controle de fluxo e operações aritméticas

## Como executar
1. Copie o arquivo `banco2.cob` para um compilador COBOL (ex.: [OnlineGDB](https://www.onlinegdb.com/online_cobol_compiler))
2. Rode o programa
